# Belldandy / Goddess Relief Office 两年期（24个月）融资计划

**总融资金额：1,000万 人民币 (约 1.4M USD)**
**核心策略：极客精简团队 + 居民楼车库创业 + 重仓创作者生态补贴**

## 📊 详细预算明细

### 一、 核心战力与极简团队（约 592 万，占比 59.2%）
保持紧凑、高战斗力阵型，坚决不招冗余人员。核心团队保持极度克制，靠期权及未来生态收益分红（90/10 计划）绑定，向投资人展示极高的资金杠杆率。

1. **核心成员薪资 (5人) —— 24个月共计 444 万**
   * **制作人/总经理**：4 万/月（统筹商业化、产品愿景与生态运转）
   * **技术总监**：4 万/月（主攻多模态大流量并发及底层重构架构研磨）
   * **主策划**：2.5 万/月（主抓 Agent 交互、方法论 SOP 与核心体验）
   * **主美**：4 万/月（把控 Neo-Mythology Tech 视觉引擎及 Canvas 设计展现）
   * **运营总监**：4 万/月（开拓 Workshop 生态与前线创作者联盟接入）
   * *小计：18.5 万/月 × 24个月 = 444 万*

2. **辅助与执行成员 (2人) —— 24个月共计 48 万**
   * 技术预备 / 社区群管 / QA（全直营远程模式），均薪 1 万/月。
   * *小计：2 万/月 × 24个月 = 48 万*

3. **社保统筹保障与基础福利**
   * 按实际薪资盘子 20% 预估刚性企业统筹（五险一金）及节日福利支出。
   * *小计：约 100 万*

---

### 二、 创作者与社区生态冷启动（重仓区，230 万，占比 23.0%）
这 **230万** 的“弹药库”是我们撬动千万级估值、构筑核心壁垒的终极武器。我们将多余资金全部加注此板块。

1. **Workshop“引路人”悬赏基金（100 万）**
   * **定向约稿与买断 (50 万)**：定向重金邀请开发者、B站知名技术 UP 主或 Notion/Obsidian 重度效率圈大佬，为 Belldandy 定制首批高质量的硬核应用场景及专业 Methodologies（如自动化金融研报生成 Agent、全自动渗透测试分析节点图等）。
   * **创作者保底池 (50 万)**：作为前期 90/10 收益分成的“平台保底补贴”。在社区初期没有 C 端付费时，平台自掏腰包补贴第一批优质创作者，确保他们有动力持续输出并长期维护高质量模块。

2. **破圈 PR 与精准技术圈投放（130 万）**
   * **硬核技术圈渗透**：精准重资赞助 V2EX、掘金、开源中国等开发者社区核心版块，合作优质技术播客（Podcast）及线下黑客马拉松，吸纳高密度极客受众。
   * **高质量 Demo 视频营销**：用 Belldandy 最具魔法感的操作流程（浏览器接管、自动修 Bug、多模态联动、Canvas 宏大重构演示），制作极具冲击力的视频并进行精准买量投流（B站、YouTube、Twitter）。我们要的是高净值开发者与重度生产力黑客，而非泛娱乐流量。

---

### 三、 服务器、多模态 API 与基建（约 96 万，占比 9.6%）
人员压缩，则系统火力必须全开，保持满配云端算力。

* **云端调用及基建池：4 万/月 × 24个月 = 96 万**
  * 包含：为新用户（注册送额度）提供的体验引流补贴（大语言模型耗时极度耗算力）。
  * 包含：`office.goddess.ai` 承接十万级高并发（Hono Node Server/Vercel等边缘节点架构）与长连接推送耗费。
  * 包含：创作者向 Workshop 不断上传超大体积模组、插件包及富媒体（图示、视频），产生巨量对象存储与全球 CDN 加速分配花销。

---

坚决落实“车库文化”，把每一分前沿开销降到冰点。

* **居民楼研发基地**：租金 3000 元/月 × 24个月 = **7.2 万**。
* **水电网及高频商宽**：2000 元/月 × 24个月 = **4.8 万**。
* **硬核重装生产力设备**：7名人员均配齐 1.5 万标准/台的高性能级开发工作站 = **10.5 万**。
* **法务及必备核心备案开支**：**10 万**。特别是大模型的生成式 API 合规双语备案、软著证书办理，以及必要的公司与财务代账代理开销。

### 五、 应急与抗风险储备金（约 49.5 万，占比 5.0%）

* 手心强握约 **50万** 现金等价物，作为雷打不动的“防守垫”，专项防范：大厂接口突发式恶性暴涨、难以预判的新发政策整改停摆冲击，亦或是流量红海时期遭受的大规模恶意流量（CC/DDoS）攻击。
