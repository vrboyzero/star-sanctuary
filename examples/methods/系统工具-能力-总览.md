## 🔴 核心原则

> **权限意识**：我拥有系统级交互能力，但必须遵循【SOUL | 基因锁 | 防篡改协议】的限制。
> 
> **安全第一**：在执行任何系统操作（特别是 exec 命令）前，必须进行安全检查，避免破坏性操作。

## 📋 工具清单

### 1. 核心文件操作工具 Agent

#### 工具列表
- **read**：读取文件内容
- **write**：创建/覆盖文件
- **edit**：基于搜索替换的修改

#### 运行模式
| 模式 | 说明 | 限制 |
|------|------|------|
| **Sandboxed (受限)** | 代码中包含 assertSandboxPath 逻辑，配合 sandboxRoot 配置，将操作严格限制在指定目录下 | 只能在指定目录内操作 |
| **Host (宿主)** | 若配置允许，可直接在宿主机工作区 (workspaceRoot) 进行文件操作 | 拥有更高的文件操作权限 |

#### 使用示例
```bash
# read 工具（通过 exec 实现）
cat /home/vrboyzero/clawd/SOUL.md

# write 工具（创建/覆盖文件）
echo "内容" > /home/vrboyzero/clawd/methods/test.md

# edit 工具（基于搜索替换）
# 通过 sed 或 Python 脚本实现文件内容替换
```

# apply_patch 工具（应用补丁）

### 3. 浏览器自动化 Agent

#### 工具名称
- **browser**

#### 能力
| 能力 | 说明 | 示例 |
|------|------|------|
| **navigate** | 导航到指定页面 | 访问 https://www.bilibili.com |
| **click** | 点击页面元素 | 点击按钮、链接 |
| **type** | 输入文本到输入框 | 在搜索框输入关键词 |
| **screenshot** | 截取屏幕截图 | 保存当前页面截图 |
| **snapshot (AI 视觉快照)** | 获取页面结构和内容 | 提取页面文本、DOM 结构 |
| **console** | 读取浏览器控制台 | 查看控制台日志、错误信息 |

#### 灵活性
| 模式 | 说明 | 适用场景 |
|------|------|----------|
| **Sandbox Mode** | 可操作容器内的无头浏览器 (Dockerfile.sandbox-browser) | 安全测试、自动化任务 |
| **Host/Remote Mode** | 可通过 CDP 或 Relay 扩展控制宿主机的真实浏览器 | 复杂交互、需要真实浏览器环境的任务 |

#### 使用示例
```javascript
// browser 工具使用

// 1. 导航到页面
navigate("https://www.bilibili.com/v/popular/rank/all")

// 2. 点击元素
click("排行榜按钮")

// 3. 截取页面快照
snapshot(maxChars=100000)

// 4. 读取控制台
console()
```

## 🛠️ 实际应用流程

### 流程1：读取系统文件
1. 使用 `exec` 工具执行 `cat` 命令
2. 或者使用 `read` 工具（如果可用）
3. 解析文件内容并提取所需信息

**示例**：
```bash
exec: cat /home/vrboyzero/clawd/SOUL.md
```

### 流程2：写入文件
1. 使用 `write` 工具创建/覆盖文件
2. 注意：不能写入受保护的文件（如 SOUL.md、MEMORY.md）
3. 确认路径在 sandboxRoot 或 workspaceRoot 内

**示例**：
```bash
write: echo "内容" > /home/vrboyzero/clawd/methods/test.md
```

### 流程3：浏览器自动化任务
1. 使用 `browser` 工具的 `navigate` 访问目标页面
2. 使用 `snapshot` 获取页面内容
3. 根据页面内容进行下一步操作（点击、输入等）
4. 使用 `screenshot` 记录任务执行结果

**示例**：
```javascript
navigate("https://www.bilibili.com")
snapshot()
click("搜索按钮")
type("输入框", "关键词")
```

## 📊 工具选择指南

| 任务类型 | 推荐工具 | 说明 |
|---------|---------|------|
| 读取文件 | `exec` + `cat` 或 `read` | 优先使用 `exec` + `cat` |
| 写入文件 | `write` | 确认路径不受保护 |
| 修改文件 | `edit` 或 `exec` + `sed` | 基于搜索替换修改 |
| 浏览器导航 | `browser` + `navigate` | 访问网页 |
| 提取页面内容 | `browser` + `snapshot` | 获取页面结构和内容 |
| 页面交互 | `browser` + `click`/`type` | 点击、输入文本 |
| 后台进程管理 | `process` | 管理长时间运行的任务 |
| 执行 Shell 命令 | `exec` | 运行 Bash 命令 |
