# EDU-curriculum-research.md - 教育课程标准检索整理方法

## 元信息
- **创建时间**: 2026-02-15
- **创建者**: 贝露丹蒂 (Belldandy)
- **适用场景**: 检索整理国家教育课程标准、学习目标、教学大纲等教育政策文档
- **关联工具**: mcp_MiniMax_web_search
- **版本**: v1.0

---

## 适用场景

1. 检索最新版义务教育课程标准（如2024年版）
2. 整理特定学段（小学/初中/高中）的学习目标
3. 对比不同学科的核心素养要求
4. 提取政策文件中的关键信息并结构化呈现

---

## 执行步骤

### Step 1: 明确检索目标
- [ ] 确定学段：小学 / 初中 / 高中 / 全学段
- [ ] 确定学科：语文 / 数学 / 英语 / 科学 / 其他
- [ ] 确定版本年份：如2024年版、2022年版
- [ ] 确定输出格式：表格 / 列表 / 对比分析

### Step 2: 构建检索关键词
```
推荐关键词组合：
- "2024年版 义务教育 课程标准 小学 语文"
- "2024年版 义务教育 数学 核心素养 学段目标"
- "小学英语 新课标 2024 学习目标"
- "义务教育 课程方案 2024 核心素养"
```

### Step 3: 执行网络检索
使用工具：`mcp_MiniMax_web_search`
- 输入构建好的关键词
- 获取权威来源（教育部官网、官方文件解读）
- 收集多源信息交叉验证

### Step 4: 信息提取与结构化

#### 4.1 提取核心框架
- 学段划分（1-2年级/3-4年级/5-6年级）
- 核心素养维度
- 学段目标对比

#### 4.2 整理输出格式
```markdown
# [学科]学习目标（[版本]）

## 一、总体框架
## 二、核心素养
## 三、学段目标对比表
## 四、能力培养重点
## 五、评价方式
```

### Step 5: 质量检查
- [ ] 信息来源是否权威可靠
- [ ] 数据是否准确（如识字量、词汇量等具体数字）
- [ ] 结构是否清晰易读
- [ ] 是否包含实用的对比表格

---

## 失败经验

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 检索结果不权威 | 关键词过于宽泛 | 加入"教育部""官方""课程标准"等限定词 |
| 版本信息混乱 | 新旧版本并存 | 明确指定年份，如"2024年版" |
| 内容过于零散 | 缺乏结构化整理 | 使用表格、分级标题统一格式 |
| 学科覆盖不全 | 一次检索范围过大 | 分学科逐一检索整理 |

## 成功案例

### 案例1: 2024版小学课程标准整理（2026-02-15）
- **任务**: 整理小学阶段语文、数学、英语三科学习目标
- **关键词**: "2024年版 义务教育 课程标准 小学 学习目标"
- **成果**: 
  - 提取了三大核心素养框架
  - 整理了三个学段的详细目标对比表
  - 涵盖了识字量、词汇量等具体指标
- **经验**: 分学科整理更清晰，使用表格对比学段差异效果最佳

## 注意事项

1. **权威性优先**: 优先使用教育部官方文件、人教社解读等权威来源
2. **版本确认**: 明确标注课程标准版本年份，避免新旧混淆
3. **数据准确**: 识字量、词汇量等数字需反复核对
4. **结构统一**: 多学科整理时保持格式一致，便于对比
5. **及时更新**: 教育政策可能调整，注意检索最新版本
